This extension is working well on Windows.

If you are using orther OS and find some functions are not working well,

you can choose which to disable or enable by the checkbox below.

Function 1, scoll tab

Hold the mouse right click, and then scroll the mousewheel to switch the Chrome tab. Scroll up to previous tab, down to next tab.

Function 2, page up/down

Hold pressing Ctrl and Shift, and then key down PageUp or PageDown. The tab will move its position with the direction. PageUp to previous tab, PageDown to next tab.

Function 3

Hold pressing Ctrl and Shift, and then key down "{" or "}". Popup tab to temporary group by key down "{". Just popup tab by key down "}".
